üì¶ How to Add to Your Website

Add this single script tag to your HTML page (before the closing </body> tag):

<script src="https://cdn.jsdelivr.net/npm/crypto-marquee-widget@latest/dist/crypto-marquee-widget.min.js"></script>

Then add a widget container anywhere in your HTML:

<div id="crypto-marquee"></div>

That's it! The widget will automatically initialize and display live crypto prices.
